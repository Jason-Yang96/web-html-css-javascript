## HTML
### 블록 요소
블록 요소는 말 그대로 블록처럼 만들어진다
### 인라인 요소
블록 요소를 포함할 수 없다
한 줄 안에 들어가서 인라인이다
너비와 높이를 가질 수 없다
블록 요소로 만들고 싶다면 display 속성을 바꿔주면 된다.
